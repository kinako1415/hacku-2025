# プロジェクト開発の必要性・社会的意義詳細分析

## 🏥 現在の医療現場での課題

### リハビリテーション分野の深刻な問題
1. **測定の主観性・非標準化**
   - 理学療法士・作業療法士による目視・触診測定に依存
   - 測定者間の個人差・経験差による結果のバラつき
   - 数値的・客観的データの不足
   - 進捗評価の曖昧性

2. **リハビリ継続性の困難**
   - 通院頻度の制約（月数回程度）
   - 自宅での自主訓練の進捗把握困難
   - 患者のモチベーション維持の困難
   - 効果的なフィードバック不足

3. **医療リソースの限界**
   - 理学療法士・作業療法士の慢性的人手不足
   - 1人当たりの診療時間制約
   - 医療費増大による経済的負担
   - 遠隔地・高齢者のアクセス困難

4. **データ記録・管理の非効率性**
   - 紙ベースの記録システム
   - データの一元管理困難
   - 長期的な傾向分析不可能
   - 医療機関間での情報共有困難

### 2. 数学的角度計算アルゴリズム
```typescript
// 3点ベクトル内積による高精度角度計算
const calculateAngle = (p1: Point, p2: Point, p3: Point): number => {
  const vec1 = { x: p1.x - p2.x, y: p1.y - p2.y };
  const vec2 = { x: p3.x - p2.x, y: p3.y - p2.y };
  
  const dot = vec1.x * vec2.x + vec1.y * vec2.y;
  const mag1 = Math.sqrt(vec1.x ** 2 + vec1.y ** 2);
  const mag2 = Math.sqrt(vec2.x ** 2 + vec2.y ** 2);
  
  return Math.acos(dot / (mag1 * mag2)) * (180 / Math.PI);
};
```
