# プロジェクト概要

## プロジェクト情報
- **プロジェクト名**: AI駆動手首・母指可動域リハビリテーションアプリ
- **パッケージ名**: ai-rehabilitation-app
- **バージョン**: 0.1.0
- **タイプ**: 医療系Webアプリケーション

## プロジェクトの目的
MediaPipe Hands APIを活用したカメラベースの手首・母指可動域自動測定システム。
リハビリテーション患者の運動療法進捗を定量的に追跡・記録することを目的とする。

## 主要機能
1. **測定機能**: AIカメラで手首と母指の可動域を正確に測定
2. **進捗確認**: 測定データの推移と詳細な統計情報を表示
3. **記録管理**: 日々のリハビリ記録をカレンダーで管理

## ターゲットユーザー
- リハビリテーション患者
- 理学療法士・作業療法士
- 医療従事者

## 測定対象の可動域
### 手首
- 掌屈 (Palmar Flexion): 0-90°
- 背屈 (Dorsal Flexion): 0-70°
- 尺屈 (Ulnar Deviation): 0-55°
- 橈屈 (Radial Deviation): 0-25°

### 母指
- 屈曲 (Flexion): 0-90°
- 伸展 (Extension): 0°
- 内転 (Adduction): 0°
- 外転 (Abduction): 0-60°

## 品質基準
- 測定精度: ±5°以内
- プライバシー: ローカルファースト設計
- データ永続化: IndexedDB使用
- 医療機器準拠: 将来的な医療機器認証準備
